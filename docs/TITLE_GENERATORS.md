# Генераторы заголовков

Система генераторов заголовков для автоматического формирования заголовков видео на основе описания из Telegram сообщений.

## Доступные генераторы

### Базовые генераторы

- **`simple`** - Простой генератор на основе имени файла
- **`date`** - Генератор с датой
- **`description`** - Генератор из описания
- **`composite`** - Композитный генератор, объединяющий несколько

### Генераторы для ЕГЭ

- **`ege_topic`** - Генератор для тем ЕГЭ
  - Формат: `Разбираем тему "Название темы"`
  - Пример: `Разбираем тему "Таблицы истинности"`
  
- **`ege_task`** - Генератор для заданий ЕГЭ
  - Формат: `Разбираем задание №X` или `Разбираем задание №X (подтип)`
  - Пример: `Разбираем задание №22 (22_4708)`
  
- **`ege_auto`** - Автоматический генератор для ЕГЭ
  - Определяет тип контента (тема или задание) и генерирует соответствующий заголовок

### Генераторы для Python

- **`python_topic`** - Генератор для тем Python
  - Формат: `Разбираем тему "Название темы"`
  - Пример: `Разбираем тему "Циклы"`
  
- **`python_task`** - Генератор для заданий Python
  - Формат: `Разбираем задание по теме "Название темы" номер X`
  - Пример: `Разбираем задание по теме "Циклы" номер 26`
  
- **`python_auto`** - Автоматический генератор для Python
  - Определяет тип контента (тема или задание) и генерирует соответствующий заголовок

## Использование

### Через фабрику

```python
from src.title_generators.factory import TitleGeneratorFactory
from src.models.video import VideoData

# Создать генератор
generator = TitleGeneratorFactory.create("ege_auto")

# Сгенерировать заголовок
video = VideoData(...)
title = generator.generate(video)
```

### Прямое использование

```python
from src.title_generators.ege_generators import EGEAutoTitleGenerator

generator = EGEAutoTitleGenerator()
title = generator.generate(video)
```

## Паттерны распознавания

### ЕГЭ

**Темы:**
- `Урок X. Название темы`
- `Задание X. Название темы`

**Задания:**
- `Задание X` / `Задание №X`
- `Разбор задания X`
- `Решение задания X`
- `Задание X_Y` (с подтипом)

### Python

**Темы:**
- `Урок. Название темы`
- `Видеоурок. Название темы`

**Задания:**
- `Задание X. Тема "Название темы"`
- `Видеоразбор задания X по теме "Название темы"`
- `Решение задания X на тему`

## Конфигурация

Генераторы можно настроить через конфигурационный файл:

```yaml
title_generator:
  type: "ege_auto"  # или "python_auto", "ege_task", и т.д.
```

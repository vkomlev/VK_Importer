## Парсер/выгрузка — реализовано

### Пустое описание, «Запись встречи» (436, 556, 594, 993 и 971–991, 994–1002)
- **Парсеры:** в `JSONParser` при пустом тексте сообщения используется описание предыдущего сообщения (`fallback_description`). В `HTMLParser` все видео в одном сообщении извлекаются через `find_all`, у всех одно описание. В `CustomExportParser` описание уже общее для всех `media_files` в сообщении.
- **БД:** добавлены `get_previous_in_folder(video_id)` и `update_description(video_id, description)` в `VideoStorage`.
- **Скрипт:** `scripts/fix_empty_descriptions_and_recalc.py` — копирует описание с предыдущей записи в том же `source_folder` для указанных ID и пересчитывает заголовки. Запуск: `python scripts/fix_empty_descriptions_and_recalc.py`.
